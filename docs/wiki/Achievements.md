# Achievements Guide

Don't Touch Purple has 37 achievements to unlock. Each achievement rewards you with Dust currency.

## Achievement Categories

### Score Milestones
- First Blood — Score your first point
- Getting Started — Score 50 points
- Warming Up — Score 100 points
- On Fire — Score 200 points
- Unstoppable — Score 500 points
- Legendary — Score 1000 points

### Survival
- Survivor — Survive 30 seconds
- Endurance — Survive 60 seconds
- Marathon — Survive 120 seconds
- Immortal — Survive 300 seconds

### Combo
- Combo Starter — Reach 5x combo
- Combo Master — Reach 10x combo
- Combo Legend — Reach 25x combo

### Boss Events
- Storm Survivor — Survive a Storm boss event
- Inversion Master — Survive an Inversion boss event
- Blackout Hero — Survive a Blackout event

### Special Cells
- Shield Bearer — Use a shield cell
- Frozen — Use a freeze cell
- Doubled — Activate a multiplier
- Healed — Use a medpack cell

### Game Modes
- Classic Player — Complete a Classic game
- Evolve Player — Start an Evolve game
- Stage 5 — Reach stage 5 in Evolve mode
- Stage 10 — Reach stage 10 in Evolve mode

### Collection
- Collector — Unlock 5 backgrounds
- Completionist — Unlock all backgrounds
- Bot Master — Use Bot Assist 10 times

### Social
- Sharer — Share your score
- Challenging — Challenge a friend
- Leaderboard — Appear on the global leaderboard

### Daily
- Daily Player — Play 3 days in a row
- Weekly Warrior — Play 7 days in a row
- Dedication — Play 30 days in a row

### Dust
- Earner — Earn 100 dust
- Spender — Spend 500 dust
- Wealthy — Accumulate 1000 dust

## Dust Rewards

Each achievement grants Dust currency which can be used to:
- Unlock animated backgrounds
- Activate Bot Assist
- Purchase power-ups in the shop
