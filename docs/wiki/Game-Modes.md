# Game Modes

## Classic Mode

Pure reflex training. 3x3 grid. Fixed difficulty. How long can you survive?

- Grid: 3x3 (fixed)
- Difficulty: Constant
- Special cells: Available
- Boss events: None
- Goal: Survive as long as possible, maximize score

## Evolve Mode

Progressive difficulty. The game gets harder the longer you survive.

### Stage Progression

| Stage | Grid | New Features |
|-------|------|-------------|
| 1 | 3x3 | Basic gameplay |
| 2 | 4x4 | Faster spawn rate |
| 3 | 5x5 | Special cells introduced |
| 4 | 6x6 | Boss events begin |
| 5 | 7x7 | Rare color modes |
| 6+ | 7x7 | Maximum difficulty |

### Boss Event Schedule

Boss events trigger at specific stage thresholds:
- First boss: Stage 4
- Frequency increases with stage
- Boss type is randomly selected from: Storm, Inversion, Blackout

### Difficulty Scaling

- Cell spawn rate increases per stage
- Purple cell frequency increases
- Special cell frequency adjusts
- Cell lifetime decreases (cells disappear faster)
- Grid expansion forces faster scanning

## Practice Mode

Practice specific boss events or difficulty levels without affecting your leaderboard score.
